# AI ReelShorts Prompt Control

面向 AI 短视频创作者的提示词创作与资产管理平台。

## 当前版本

MVP 第一阶段已包含：

- 提示词创建工作台
- 引导式选择流程
- 角色资产库
- 提示词存储库
- 预设提示词库
- 搜索与标签管理
- 生成结果编辑、复制、保存
- 预设版本记录
- 日夜模式切换
- 项目风格母版：统一画面基调、色彩逻辑、光影偏好、镜头语言和默认负面提示词
- 存储库智能召回：根据工作台输入、角色、情绪和光影关键词自动推荐可参考词条
- 模型配置占位与本地配置
- 电影静帧光影提取模板
- ARRI / Cooke 写实电影机参数预设

## 光影资产规范

后续光影提示词统一按以下结构输出：

```text
光影氛围：
色彩逻辑：
曝光与调色：
场景氛围：
叙事氛围：
电影机参数：
全局质感：
负面约束：
```

详细规则见 [docs/lighting-extraction-rules.md](docs/lighting-extraction-rules.md)。

## 使用方式

直接打开 `index.html` 即可使用。数据会保存在浏览器 `localStorage` 中，可通过左侧“导出”按钮备份。

如果之前已经打开过旧版本网站，新加入的默认光影资产和微表情词条会自动补入本地存储库，不会覆盖你自己保存的资产。

## 存储库与微表情资产

存储库用于保存每次调用 AI 时可参考的专业词条。微表情类型采用“总纲词条 + 单独情绪词条”的结构：总纲负责统一拆解规则，单独词条负责具体场景调用和搜索复用。

当前已内置：

- 微表情拆解总纲（参考飞书人物表情提示词文档）
- 喜悦类微表情类型库
- 哀伤类微表情类型库
- 淡漠与复杂类微表情类型库
- 愤怒类微表情类型库
- 震惊错愕类微表情类型库
- 愤怒但克制的微表情
- 委屈但不失控的微表情
- 震惊后的短暂停滞
- 怀疑试探的微表情
- 冷漠疏离的微表情
- 崩溃边缘但强行维持

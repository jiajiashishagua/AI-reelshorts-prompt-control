# 电影静帧光影资产提取规则

## 核心规律

这些光影提示词不是单纯描述“亮不亮”，而是把画面拆成可复用的电影资产：

1. 先定义场景和时间：荒漠黄昏、午后办公室、高档公寓、春日庄园。
2. 再定义主光源：日落侧逆光、落地窗漫射光、高窗斜射光、春日自然光。
3. 继续描述光线方向和质感：侧方、侧逆、低角度、漫射、柔光、丁达尔光、无生硬阴影。
4. 明确阴影形态：长阴影、条状光影、细腻阴影、干净过渡。
5. 建立色彩逻辑：暖橙金色加蓝调时刻、冷白米灰、灰绿米黄暖棕、青草绿加天空蓝。
6. 给出曝光和调色参数：色温、白平衡、对比度、曝光、高光、阴影、饱和度、清晰度。
7. 绑定电影机参数：ARRI Amira、Cooke 35mm、T1.4、24fps、EI/ASA800、Log-C、ARRIRAW。
8. 最后补叙事氛围：孤寂宿命、专业疏离、慵懒温馨、清新复古。

## 标准输出结构

后续从电影静帧中提取光影提示词时，统一输出：

```text
光影氛围：
色彩逻辑：
曝光与调色：
场景氛围：
叙事氛围：
电影机参数：
全局质感：
负面约束：
```

## 提取问题清单

看到一张电影静帧时，依次回答：

1. 这是什么场景、什么时间、什么天气或空间环境？
2. 主光源在哪里，是自然光、窗光、日落、天空光还是人工光？
3. 光线是硬光还是柔光，是正面、侧方、侧逆、顶光还是低角度斜射？
4. 阴影是长、短、硬、软、条状、块状，还是几乎没有阴影？
5. 画面主色是什么，辅助色是什么，冷暖关系如何？
6. 饱和度、对比度、曝光、高光、阴影分别偏高还是偏低？
7. 构图、景深、颗粒、清晰度和写实程度是什么风格？
8. 这组光影在讲什么情绪和故事？
9. 需要保留哪些电影机参数，哪些参数要按场景变化？
10. 哪些视觉错误必须禁止？

## 固定相机基底

除非另有要求，光影提示词默认使用：

```text
写实真人电影画质；阿莱艾美拉摄影机拍摄，竖屏构图（Vertical composition），极具情绪张力与物理细节真实感；raw photo, film grain, 8K, no CGI, no rendering, no smooth surface；24fps，EI/ASA 800，Log-C伽马，库克35mm定焦镜头S8/i系列，光圈T1.4，快门角度180°，记录格式ProRes 4444 XQ / ARRIRAW。
```

白平衡和色温按场景调整：

- 荒漠黄昏：8500K 蓝调时刻白平衡。
- 时尚办公室：5500K，整体冷白干净。
- 美式公寓：4000-4800K 或 4200K 偏冷。
- 英伦庄园：5600K 春日自然光。
